# Unlimted Hand Shell

## コマンドの種類
コマンドは引数の数で分類できる。

## コマンドの構成
`コマンド名`+`空白`+`引数`...+`改行文字`

## 実装されるコマンド

### 振動モータに関するもの
+ vbr_on:振動モーターを駆動
+ vbr_off:振動モータを停止
+ vbr [振動時間]:指定した時間の間振動する。

### その他
+ help:コマンドを一覧表示する
+ dummy:擬似的にblue-ninjaの出力を再現する
+ ack:ackをかえす

### 位置情報等
+ acc:加速度


### 電撃
+ ems [チャンネル] [時間] [電圧] [鋭さ]　電気ショックを与える。
チャンネル[0..7],時間[0..200],電圧[0..12],鋭さ[5..20]であり、内部で修正をかけている。
